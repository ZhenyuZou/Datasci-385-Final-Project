# DATASCI Final Project

This project analyzes World Bank World Development Indicators (WDI) data, focusing on population dynamics. The workflow combines SQL for data preparation and Python for analysis and visualization, with results presented in a Quarto report.

## Project Structure

- `index.qmd`: Main Quarto report
- `index.html`: Rendered HTML report
- `index.pdf`: Rendered PDF report
- `wdi_population.db`: Database file
- `Datasci385_Final_Project.ipynb`: Analysis notebook
- `_quarto.yml`: Quarto configuration

## How to Run

1. Open the project in VS Code.
2. Ensure Python and Quarto are installed.
3. Run: quarto render
4. Open `index.html` or `index.pdf` to view the report  

## Data Source
World Bank World Development Indicators (WDI), including:
- Life expectancy  
- Under-5 mortality rate  
- Adolescent fertility rate  

## Notes
The project explores trends over time and relationships between key population indicators across countries.
